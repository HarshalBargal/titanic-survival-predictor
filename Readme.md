# Titanic Survival Predictor 🚢

This is a simple Streamlit app that predicts Titanic survival using logistic regression.  
It also includes visualizations like countplots, barplots, and pie charts.

## 📂 Files

- `app.py` — Streamlit app code
- `train.csv` — Titanic dataset
- `titanic_model.pkl` — Pre-trained model
- `requirements.txt` — Python libraries to run the app

## 🚀 How to Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
